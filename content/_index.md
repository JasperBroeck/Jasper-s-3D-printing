# About Me
Ik ben Jasper, een vrolijke jonge gast. Sinds ik een 3D printer heb, print ik wat ik maar nodig heb en ik wil anderen ook net dat plezier kunnen geven. Daarom start ik deze printservice. Je plaatst je bestelling en enkele dagen later heb je je stuk in huis!

Hoe ga ik te werk?

1. Je plaatst je bestelling
2. Ik controleer het model en bepaal de prijs met deze {{< term "formule" >}}
3. Ik stuur je een betaalmailtje
4. Na betaling start ik de print
5. Na afwerking kun je deze komen halen of verzend ik deze, verzending voor jouw rekening.
